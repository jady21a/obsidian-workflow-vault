# 🎨 11.Excalidraw — 手绘 / 白板

用 **Excalidraw 插件**画图、做白板、画思维导图的地方。

**用法**
- 命令面板 → `Excalidraw: Create new drawing`，新建的 `.excalidraw.md` 默认就会落在本目录。
- `assets/` 放导出的 PNG/SVG；`before/` 可放旧版本归档。
- 画好的图可以用 `![[图名.excalidraw]]` 嵌入任何笔记。

> [!note]
> 本目录暂未放示例图（二进制画板不适合纯文本示例）。装上 Excalidraw 插件后新建一张即可，结构已就位。
