# 🤖 12.skills — AI / 工作流技能包

存放可复用的 **Claude / AI 技能（skill）**：每个技能是一个文件夹，里面用 `SKILL.md` 描述「什么时候用、怎么用」，配套 `scripts/`、`references/`、`evals/`。

**典型结构**
```
某技能/
├── SKILL.md        # 触发条件 + 使用说明（必需）
├── scripts/        # 配套脚本
├── references/     # 参考资料 / 模板
└── evals/          # 测试用例
```

**用法**
- 把重复性的 AI 工作流（翻译字幕、写脚本、回评论…）沉淀成技能，下次一句话触发。
