# 📖 2.Read — 读书 · 观影

整个「输入消化」区。读书、看电影 / 剧集的笔记都在这里，靠 frontmatter 字段 + Dataview 自动汇总成书架、观影库。

**子目录**
- `books/` —— 书籍笔记（封面图放 `books/assets/`）
- `media-DB/movie`、`media-DB/series` —— 用 Media DB 插件抓取的影视条目
- `douban/` —— 豆瓣插件抓取的条目（book / movie / teleplay）

**两个入口页**
- [[书架]] —— 所有书的 Dataview 总览
- [[观影库]] —— 所有影视的 Dataview 总览
- [[bases书架.base]]——用bases来展示的书架
注:dataview展示的方式虽然美观,但是页面卡顿的问题也相对严重,后续我准备用bases来展示,调整好之后分享
